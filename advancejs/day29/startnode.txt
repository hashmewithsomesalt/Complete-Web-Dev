To create a server, we need node HTTP module.
Http module: Allows data to be transferred over the HTTP protocol. Create servers. Handle requests from clients. Provide responses to those requests.

The response object: Has methods which allow us to 
=> specify content-type
=> set status codes (200, 400 etc)
=> provide content (html, JSON, images)

The Request-Response Cycle:
Client sends HTTP request:
Request: Method, request, data
Server receives HTTP request: handle request: filtering data, throwing an error, generate a response
Replies with an HTTP response: resource (json), content-type(application/json), status code, status message
Client receives it

The request object gives us access to the incoming request. It contains:
The URL the client used, The headers, Any data sent, The method (GET, POST, DELETE)

The methods tells us what type of request is asked by client: get, post, delete, put, patch

JSON: When rest api sends data, they commonly uses JSON data
HTTP is a text-based protocol. All data transferred between the client and the server must be in the form of strings.
JSON.stringify(<data-to-convert>)

Content-Types: application/json, text/html, text/css, application/javascript
