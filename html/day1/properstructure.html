Proper Structure

This is an example of a properly structured HTML document.

Headings

Headings are used to organize content and create a hierarchy. Use <h1> for the main title, <h2> for subheadings, and so on.

Paragraphs

Paragraphs are used to group related sentences together. Use the <p> tag to create a paragraph.

Lists

Lists are used to organize items. Use the <ul> tag for unordered lists and the <ol> tag for ordered lists.
